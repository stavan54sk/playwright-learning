## 🛠 How to Use

1. Clone the repository:
    ```bash
    git clone https://github.com/stavan54sk/playwright-learning.git
    cd playwright-learning
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start exploring the chapters and practicing with the provided examples.
